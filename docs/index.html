MaxBox

A native macOS Gmail client built with SwiftUI. Fast, private, keyboard-driven.

macOS 14+ SwiftUI Gmail REST API MIT License
View on GitHub
📨

Multi-Account

Add multiple Gmail accounts and browse them in a unified "All Accounts" view or individually.

🗗

Multi-Window

Compose in separate windows, open messages in their own window, run multiple main windows side by side.

🔍

Instant Search

Animated expandable search bar with instant filtering across your messages.

✏️

Compose & Drafts

Rich text editor, auto-save drafts, and resume editing by double-clicking any draft.

⚡

Background Caching

Messages are cached locally after account creation for fast, responsive access.

🔒

Keychain Security

OAuth tokens are stored in the macOS Keychain. No passwords are ever saved to disk.

Quick Start

# Clone the repo
git clone https://github.com/pubino/maxbox.git && cd maxbox

# Set up GCP OAuth credentials (interactive)
./scripts/gcp-setup.sh

# Build and run
source scripts/.env.local && ./scripts/run.sh
